## Example


```javascript
import epochSeconds from "epoch-seconds"

const result = epochSeconds()
```

Variable `result` will be:

```javascript
1549410770
```












## Development



Setting up:
```bash
git clone git@github.com:jaid/epoch-seconds.git
cd epoch-seconds
npm install
```
Testing:
```bash
npm run test:dev
```
Testing in production environment:
```bash
npm run test
```
